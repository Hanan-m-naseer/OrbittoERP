# 🌀 OrbitoERP

**OrbitoERP** is a modular ERP (Enterprise Resource Planning) system designed for small–medium businesses.  
It covers core business functions like user & company management, inventory, sales, purchases, and basic finance reporting.

This repository will be the backend foundation of **OrbitoERP** with clean architecture, REST APIs, and role-based access control.

---

## 🚀 Features (Planned)

- Company & profile management  
- User management with roles (Owner, Admin, Manager, Employee)  
- Inventory management (products, categories, stock logs)  
- Sales & purchase modules  
- Dashboard & reports  
- Basic finance summaries  
- Clean API design (future)
